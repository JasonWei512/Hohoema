# Hohoema

HohoemaはオープンソースなWindows 10向けのニコニコ動画プレイヤーです。

現在アプリストアに申請中です。（2016/08/02）

開発者がARM端末を持っていないためARM版はアプリストア掲載の時期未定です。

## 主な機能

* 動画再生とコメントの投稿
* ランキングや検索から動画を見つける
* 動画タイトルやコメントのNG設定
* お気に入りした「ユーザー」「タグ」「マイリスト」が作る動画フィードを「お気に入り動画一覧」として表示
* キャッシュ管理機能

## 実装予定

v0.4 プレイリスト機能
